check_grammar {gramr}R Documentation

check_grammar

Description

Checks a piece of text for common grammatical errors and returns a list of suggestions.

Usage

check_grammar(text, options = list())

Arguments

text

The text to check

options

Which rules should be ignored see https://github.com/btford/write-good for details.

Value

A character vector of suggestions for grammar changes.

Examples


check_grammar("So the cat was stolen.", list(passive = FALSE))
