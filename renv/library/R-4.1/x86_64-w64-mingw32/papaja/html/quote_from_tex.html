quote_from_tex {papaja}R Documentation

Quote from TeX document

Description

Includes a labelled quote from a LaTeX document 'asis'.

Usage

quote_from_tex(x, file)

Arguments

x

Character. One or more quote labels.

file

Character. Path to LaTeX file from which to quote.

Details

Searches the LaTeX document for a labelled quotes preceeded and followed by % <@~{#quote-label} and % ~@> tags in LaTeX comments
